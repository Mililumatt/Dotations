# GUIDE SIGNATURE MOBILE (QR)

## 1. ETAT ACTUEL

- En local, le lien QR utilise l'URL du reseau local (ex: `http://10.8.0.23:8123`).
- Le smartphone doit etre sur le meme Wi-Fi que le PC.

## 2. MODE PUBLIC (FUTUR HEBERGEMENT)

- Aller dans `BASES DE REFERENCE`.
- Dans la carte `SIGNATURE MOBILE (FLASHCODE)`, saisir l'URL publique.
- Exemple : `https://dashboard-dotations.mon-domaine.fr`
- Cliquer `ENREGISTRER`, puis `SAUVEGARDER`.

## 3. RESULTAT ATTENDU

- Les boutons `SIGNER SUR TELEPHONE` (document arrivee/sortie) generent un lien/QR base sur l'URL publique.
- Le smartphone peut signer en 4G/5G ou en Wi-Fi.
- La signature remonte automatiquement dans la fiche du personnel.

## 4. CHECK RAPIDE

1. Ouvrir un document arrivee ou sortie avec une personne selectionnee.
2. Cliquer `SIGNER SUR TELEPHONE`.
3. Scanner le QR avec le telephone.
4. Signer et valider.
5. Verifier sur PC : statut signature passe en signe et date mise a jour.
